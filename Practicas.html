Ejemplos de la propiedad position

position absolute: determinada por las coordenadas en el documento

position relative: cualquiera de los elementos hijos se posicionan de forma relativa al padre

position fixed: coordenadas relativas a la pantalla, fija en el viewport

position sticky: deja su posición siempre que puede dentro del contenedor

Ejemplos de z-index y apilamiento

El z-index es una propiedad de CSS que controla el orden de apilamiento de los elementos posicionados (con position: relative, absolute, fixed o sticky). Indica qué elemento se muestra por encima de otro cuando se superponen.

GRID

CSS Grid es un sistema de diseño bidimensional que te permite crear layouts complejos y responsivos de manera sencilla, organizando los elementos en filas y columnas.

JavaScript

JavaScript es un lenguaje de programación esencial para el desarrollo web moderno. Permite añadir interactividad, dinamismo y funcionalidades avanzadas a las páginas web, como responder a eventos del usuario, modificar el contenido de la página en tiempo real y comunicarse con servidores. Gracias a JavaScript, las páginas web dejan de ser estáticas y se convierten en aplicaciones interactivas y atractivas para los usuarios.

Botones

Botón que cambia el color del fondo: document.body.style.backgroundColor = "#E9C3DE";
(este botón no cambia el color permanentemente, recarga la página para volver al color original)

Botón que te lleva al inicio de la página: window.scrollTo({ top: 0, behavior: 'smooth' });

Botoón que abre una ventana emergente con un mensaje: alert('Hola, soy un botón');
